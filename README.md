# Python Basic Programs – Internship Tasks

## Overview
This repository contains Python programs completed as part of internship tasks. The programs demonstrate fundamental Python concepts such as arithmetic operations, string manipulation, data structures, loops, pattern generation, object-oriented programming, and basic data analysis operations.

## Project Files

Task_1py.ipynb  
Contains programs related to arithmetic operations, string manipulation, and basic Python concepts.

Task_2py.ipynb  
Contains programs related to lists, tuples, dictionaries, conditional statements, loops, and pattern programs.

DataFrame.ipynb  
Implements DataFrame-like operations using core Python data structures such as lists and dictionaries without using external libraries.

Dataframe_with_pandas.ipynb  
Demonstrates DataFrame creation and operations using the pandas library for structured data handling.

Numpy_with_pandas.ipynb  
Shows how NumPy operations can be performed on pandas DataFrames for numerical data processing.

## Technologies Used
- Python 3
- pandas
- NumPy

## Learning Outcomes
- Understanding Python programming fundamentals
- Working with lists, tuples, dictionaries, and strings
- Implementing loops and conditional logic
- Understanding basic object-oriented programming
- Learning basic data analysis concepts using DataFrames
- Applying NumPy operations on structured data

## Conclusion
These internship tasks helped strengthen foundational Python programming knowledge and introduced basic data analysis concepts using DataFrames and numerical operations.
