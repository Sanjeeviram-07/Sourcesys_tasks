# Python Basic Programs – Internship Tasks

## Overview
This repository contains a collection of basic Python programs completed as part of an internship assignment. The programs focus on fundamental programming concepts and are designed to strengthen problem-solving skills and understanding of Python syntax. The tasks cover essential areas of Python programming including arithmetic operations, string handling, data structures, control flow statements, loops, pattern generation, and object-oriented programming.

## Topics Covered

### Arithmetic Operations
Programs related to performing mathematical calculations such as basic arithmetic operations, power calculation, and simple interest computation.

### String Manipulation
Programs demonstrating different string operations including case conversion, counting characters, finding length, replacing words, reversing strings, and checking palindromes.

### List Operations
Programs that use Python lists to perform tasks such as adding elements, inserting elements at specific positions, sorting lists, reversing lists, removing elements, finding maximum values, calculating sums, and removing duplicate elements.

### Tuple Operations
Programs that illustrate tuple usage including accessing elements, tuple length calculation, tuple unpacking, counting elements, finding index positions, nested tuples, and tuple concatenation.

### Dictionary Operations
Programs demonstrating dictionary concepts such as accessing values, adding and removing key-value pairs, checking the existence of keys, and retrieving keys and values from dictionaries.

### Conditional Statements
Programs using decision-making structures such as if, elif, and else to perform logical checks like identifying number types, evaluating grades, determining leap years, and other conditional operations.

### Loop Programs
Programs that utilize loops such as for and while loops to perform repeated operations including number printing, factorial calculations, multiplication tables, and summation of numbers.

### Pattern Programs
Programs using nested loops to generate different patterns such as square patterns, triangle patterns, and number-based patterns.

### Object-Oriented Programming
Programs demonstrating basic object-oriented programming concepts such as classes, objects, constructors, inheritance, encapsulation, and polymorphism.

## Technologies Used
- Python 3

## Learning Outcomes
- Understanding Python syntax and structure
- Implementing basic algorithms using Python
- Working with Python data structures
- Applying conditional logic and loops
- Understanding basic object-oriented programming concepts
- Developing problem-solving and logical thinking skills

## Conclusion
The internship tasks helped in strengthening foundational Python programming knowledge by practicing various fundamental concepts. Completing these programs improved coding proficiency and provided practical exposure to essential programming techniques used in software development.
